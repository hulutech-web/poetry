#### 1.1 项目介绍
古诗文应用是一款基于wails的跨平台桌面应用。  
- 支持windows,mac 
- 旨在为广大用户提供便捷的古诗文阅读、学习、欣赏和分享服务。
- 应用集成了语音服务。
- 支持一键复制，和分享自己喜欢的古诗文，提高自己的文学素养和审美品味。
- 安装程序见：https://github.com/hulutech-web/poetry/releases
